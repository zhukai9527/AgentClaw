# arXiv Draft: Memory as a Control System for Production AI Agents

This folder contains the arXiv-ready source package for the AgentClaw memory paper.

Files:

- `main.tex` - paper source
- `references.bib` - BibTeX references
- `arxiv-metadata.md` - metadata to paste into arXiv

Recommended upload package:

```bash
cd docs/papers/arxiv-memory-control-system
zip agentclaw-memory-control-arxiv.zip main.tex references.bib
```

Suggested arXiv category:

- Primary: `cs.SE`
- Cross-list: `cs.AI`

Important:

arXiv expects authors to self-submit from a registered author account. Use this draft as the source package, then verify title, abstract, author identity, license, and category in the arXiv submission UI before the final `Submit Article` step.
